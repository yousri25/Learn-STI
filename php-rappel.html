FonctionDescriptionExemple

④Fonctions sur les chaînes de caractères

FonctionDescriptionExemple
chr($code)Retourne le caractère dont le code ASCII est passé comme argument.chr(65) → "A"
ord($ch)Retourne le code ASCII du 1er caractère de l'argument.ord("A") → 65
strlen($ch)Retourne la longueur d'une chaîne.strlen("Bonjour") → 7
substr($ch, $d, $lg)Retourne une sous-chaîne à partir de la position d de longueur lg.substr("Bonjour",0,3) → "Bon"
strpos($ch, $ch1)Retourne la position de la première occurrence du 2ème argument dans le 1er.strpos("Bonjour","o") → 1
strcmp($ch1, $ch2)Retourne le résultat de la comparaison de deux chaînes (<0, 0, >0).strcmp("a","b") → -1
str_replace($ch1,$ch2,$ch)Remplace toutes les occurrences d'un motif par un autre dans une chaîne.str_replace("o","0","Bonjour") → "B0nj0ur"
strtolower($ch)Convertit tous les caractères d'une chaîne en minuscule.strtolower("HELLO") → "hello"
strtoupper($ch)Convertit tous les caractères d'une chaîne en majuscule.strtoupper("hello") → "HELLO"
trim($ch)Supprime les espaces au début et à la fin de la chaîne.trim(" Bonjour ") → "Bonjour"

⑨Fonctions PHP pour MySQL (MySQLi)*

* Utiliser les fonctions fournies par l'extension MySQLi dans son interface procédurale.
FonctionDescription
mysqli_connect($host,$user,$pwd,$db) Établit une connexion à une base de données MySQL. Retourne un objet connexion ou false en cas d'échec.
mysqli_query($conn, $sql) Envoie une requête SQL à la base de données. Retourne un résultat pour SELECT, ou true/false pour les autres requêtes.
mysqli_close($conn) Ferme la connexion à la base de données.
mysqli_fetch_array($result) Retourne une ligne de résultat sous forme de tableau associatif et numérique — accès par nom de colonne ET par index.
mysqli_fetch_row($result) Retourne une ligne de résultat sous forme de tableau indexé numériquement — accès par index uniquement.
mysqli_num_rows($result) Retourne le nombre de lignes dans un résultat de requête SELECT.
mysqli_affected_rows($conn) Retourne le nombre de lignes affectées par la dernière requête SQL exécutée (INSERT, UPDATE, DELETE).
mysqli_error($conn) Retourne un message d'erreur généré par la dernière requête SQL exécutée.

💡Exemples complets — PHP & MySQLi

Ces exemples utilisent MySQLi en interface procédurale pour interagir avec une base de données ecole contenant une table eleve (id, nom, prenom, note).
① Connexion à la base de donnéesmysqli_connect · die
<?php // Paramètres de connexion $host = "localhost"; $user = "root"; $pwd = ""; $db = "ecole"; // Établir la connexion $conn = mysqli_connect($host, $user, $pwd, $db); // Vérifier si la connexion a réussi if (!$conn) { die("Erreur de connexion : " . mysqli_connect_error()); } echo "Connexion réussie !"; ?>
② Sélectionner et afficher des données (SELECT)mysqli_query · mysqli_fetch_array · mysqli_num_rows
<?php $conn = mysqli_connect("localhost", "root", "", "ecole"); if (!$conn) die(mysqli_connect_error()); // Requête SELECT $sql = "SELECT id, nom, prenom, note FROM eleve ORDER BY nom ASC"; $result = mysqli_query($conn, $sql); // Vérifier le nombre de lignes retournées if (mysqli_num_rows($result) > 0) { // Parcourir chaque ligne du résultat while ($ligne = mysqli_fetch_array($result)) { echo "ID : " . $ligne["id"] . " | "; echo "Nom : " . $ligne["nom"] . " "; echo $ligne["prenom"] . " | "; echo "Note : " . $ligne["note"] . "<br>"; } } else { echo "Aucun enregistrement trouvé."; } mysqli_close($conn); ?>
③ Afficher les résultats dans un tableau HTMLSELECT + while + tableau HTML
<?php $conn = mysqli_connect("localhost", "root", "", "ecole"); if (!$conn) die(mysqli_connect_error()); $sql = "SELECT nom, prenom, note FROM eleve WHERE note >= 10"; $result = mysqli_query($conn, $sql); ?> <table border="1"> <tr><th>Nom</th><th>Prénom</th><th>Note</th></tr> <?php while ($row = mysqli_fetch_array($result)) : ?> <tr> <td><?php echo $row["nom"]; ?></td> <td><?php echo $row["prenom"]; ?></td> <td><?php echo $row["note"]; ?></td> </tr> <?php endwhile; ?> </table> <?php mysqli_close($conn); ?>
④ Insérer des données (INSERT)$_POST · isset · mysqli_query · mysqli_affected_rows
<!-- Formulaire HTML (methode POST) --> <form method="POST" action="ajouter.php"> <input type="text" name="nom" placeholder="Nom" required> <input type="text" name="prenom" placeholder="Prénom" required> <input type="number" name="note" placeholder="Note" min="0" max="20"> <input type="submit" value="Ajouter"> </form> <!-- ajouter.php --> <?php if (isset($_POST["nom"])) { $conn = mysqli_connect("localhost", "root", "", "ecole"); if (!$conn) die(mysqli_connect_error()); // Récupérer les valeurs du formulaire $nom = $_POST["nom"]; $prenom = $_POST["prenom"]; $note = $_POST["note"]; // Requête INSERT $sql = "INSERT INTO eleve (nom, prenom, note) VALUES ('$nom', '$prenom', '$note')"; if (mysqli_query($conn, $sql)) { echo mysqli_affected_rows($conn) . " ligne(s) insérée(s) avec succès !"; } else { echo "Erreur : " . mysqli_error($conn); } mysqli_close($conn); } ?>
⑤ Modifier des données (UPDATE)$_POST · UPDATE · mysqli_affected_rows
<!-- Formulaire de modification --> <form method="POST" action="modifier.php"> <input type="hidden" name="id" value="3"> <input type="number" name="note" placeholder="Nouvelle note" min="0" max="20"> <input type="submit" value="Modifier"> </form> <!-- modifier.php --> <?php if (isset($_POST["id"])) { $conn = mysqli_connect("localhost", "root", "", "ecole"); if (!$conn) die(mysqli_connect_error()); $id = $_POST["id"]; $note = $_POST["note"]; // Requête UPDATE $sql = "UPDATE eleve SET note = '$note' WHERE id = '$id'"; if (mysqli_query($conn, $sql)) { echo mysqli_affected_rows($conn) . " ligne(s) mise(s) à jour !"; } else { echo "Erreur : " . mysqli_error($conn); } mysqli_close($conn); } ?>
⑥ Supprimer des données (DELETE)$_GET · DELETE · mysqli_affected_rows
<!-- Lien de suppression (methode GET) --> <a href="supprimer.php?id=3">Supprimer l'élève #3</a> <!-- supprimer.php --> <?php if (isset($_GET["id"])) { $conn = mysqli_connect("localhost", "root", "", "ecole"); if (!$conn) die(mysqli_connect_error()); $id = $_GET["id"]; // Requête DELETE $sql = "DELETE FROM eleve WHERE id = '$id'"; if (mysqli_query($conn, $sql)) { echo mysqli_affected_rows($conn) . " élève(s) supprimé(s) !"; } else { echo "Erreur : " . mysqli_error($conn); } mysqli_close($conn); } ?>
⑦ Bonne pratique — Fichier de connexion séparérequire()
// ── connexion.php ───────────────────────────────── <?php $conn = mysqli_connect("localhost", "root", "", "ecole"); if (!$conn) die("Connexion impossible : " . mysqli_connect_error()); ?> // ── index.php ────────────────────────────────────── <?php require("connexion.php"); // inclut et exécute connexion.php $sql = "SELECT * FROM eleve"; $result = mysqli_query($conn, $sql); echo "Nombre d'élèves : " . mysqli_num_rows($result); mysqli_close($conn); ?>
⑧ Récapitulatif CRUD — Toutes les opérations en un seul scriptSELECT · INSERT · UPDATE · DELETE
<?php // ── 1. Connexion ────────────────────────────────── $conn = mysqli_connect("localhost", "root", "", "ecole"); if (!$conn) die(mysqli_connect_error()); // ── 2. INSERT ───────────────────────────────────── $sql = "INSERT INTO eleve (nom, prenom, note) VALUES ('Trabelsi','Amine',15)"; mysqli_query($conn, $sql); echo mysqli_affected_rows($conn) . " ligne(s) insérée(s)<br>"; // ── 3. UPDATE ───────────────────────────────────── $sql = "UPDATE eleve SET note = 18 WHERE nom = 'Trabelsi'"; mysqli_query($conn, $sql); echo mysqli_affected_rows($conn) . " ligne(s) modifiée(s)<br>"; // ── 4. SELECT ───────────────────────────────────── $sql = "SELECT nom, prenom, note FROM eleve WHERE note >= 10"; $result = mysqli_query($conn, $sql); echo mysqli_num_rows($result) . " élève(s) admis :<br>"; while ($r = mysqli_fetch_array($result)) { echo $r["nom"] . " " . $r["prenom"] . " → Note : " . $r["note"] . "<br>"; } // ── 5. DELETE ───────────────────────────────────── $sql = "DELETE FROM eleve WHERE note < 5"; mysqli_query($conn, $sql); echo mysqli_affected_rows($conn) . " élève(s) supprimé(s)<br>"; // ── 6. Fermeture ────────────────────────────────── mysqli_close($conn); ?>
⑨ Recherche avec $_GET et isset()$_GET · isset · LIKE
<!-- Formulaire GET (recherche) --> <form method="GET" action="recherche.php"> <input type="text" name="nom" placeholder="Rechercher un élève..."> <input type="submit" value="Chercher"> </form> <!-- recherche.php --> <?php $conn = mysqli_connect("localhost", "root", "", "ecole"); if (!$conn) die(mysqli_connect_error()); if (isset($_GET["nom"]) && $_GET["nom"] != "") { $nom = $_GET["nom"]; $sql = "SELECT * FROM eleve WHERE nom LIKE '%$nom%'"; $result = mysqli_query($conn, $sql); if (mysqli_num_rows($result) > 0) { while ($r = mysqli_fetch_array($result)) { echo $r["nom"] . " " . $r["prenom"] . " — Note : " . $r["note"] . "<br>"; } } else { echo "Aucun résultat pour : " . $nom; } } else { echo "Veuillez entrer un nom à rechercher."; } mysqli_close($conn); ?>
