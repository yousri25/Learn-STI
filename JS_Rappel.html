Apprendre JavaScript

Un guide pas à pas pour rendre tes pages web interactives et dynamiques

Chapitre 1 · Introduction

C'est quoi JavaScript ?

HTML structure, CSS stylise — mais qui fait bouger la page ? C'est le rôle de JavaScript.

// Définition

JavaScript (JS) est un langage de programmation qui s'exécute dans le navigateur. Il permet de rendre les pages web interactives et dynamiques : réagir aux clics, modifier le contenu, valider des formulaires, afficher des alertes, et bien plus encore.

💡 Rappel du trio web

HTML = les murs, les portes (la structure).
CSS = la peinture, les meubles (l'apparence).
JavaScript = l'électricité, la domotique (le comportement — tout ce qui réagit et se déplace).

Comment intégrer JavaScript dans une page
JS Inline dans la balise
Directement dans un attribut d'événement HTML. Pratique pour des actions simples, déconseillé en production.
<button onclick="alert('Bonjour')">Clic</button>
JS Interne balise <script>
Dans une balise <script> à la fin du <body>. Pratique pour une seule page.
<script>
  // code ici
</script>
JS Externe fichier .js ✅
Dans un fichier script.js lié avec l'attribut src. Méthode recommandée — séparation du code.
<script src="script.js"></script>
Les premières instructions
Avant d'aller plus loin, voici les deux façons les plus simples d'afficher un résultat en JavaScript.
script.js
// Affiche une boîte de dialogue
alert("Bonjour le monde !");

// Affiche directement dans le document HTML
document.write("<h2>Bienvenue !</h2>");

// Affiche dans la console développeur (F12)
console.log("Message de debug");

// Modifie le contenu d'un élément HTML via son id
document.getElementById("resultat").innerHTML = "Nouveau contenu";

Chapitre 2 · Les données

Variables & Types de données

Une variable est une boîte qui stocke une valeur. En JS, on la déclare avec var, let ou const.

Déclarer une variable
variables.js
// var — ancienne façon (portée fonction)
var prenom = "Ali";

// let — moderne, peut être modifié (portée bloc)
let age = 17;
age = 18; // ✅ modification autorisée

// const — constante, ne peut pas être modifiée
const PI = 3.14;
// PI = 3; ❌ erreur !
Les types de données
TypeDescriptionExemple
StringChaîne de caractères — toujours entre guillemets"Bonjour" · 'Ali'
NumberNombre entier ou décimal17 · 3.14
BooleanValeur logique — vrai ou fauxtrue · false
ArrayTableau — liste de valeurs indexées à partir de 0[10, 20, 30]
ObjectObjet — paires clé : valeur{nom: "Ali", age: 17}
nullValeur vide intentionnellelet x = null;
undefinedVariable déclarée mais sans valeurlet y;
Conversion de types
JS est un langage typé dynamiquement : une variable peut changer de type. Mais il faut parfois convertir explicitement une valeur d'un type à un autre.
Number(ch)
Convertit une chaîne en nombre. Si impossible, retourne NaN.
Number("42") → 42
Number("abc") → NaN
parseInt(ch [,b])
Convertit en entier dans la base b. Ignore la partie décimale.
parseInt("3.9") → 3
parseInt("FF",16) → 255
parseFloat(ch)
Convertit une chaîne en nombre réel (décimal).
parseFloat("3.14") → 3.14
String(a)
Convertit n'importe quelle valeur en chaîne de caractères.
String(42) → "42"
String(true) → "true"
isNaN(ch)
Retourne true si la valeur n'est pas un nombre convertible.
isNaN("abc") → true
isNaN("42") → false
prompt()
Affiche une boîte de dialogue pour saisir une valeur. Retourne toujours une chaîne.
let n = prompt("Entrez un nb:");
n = Number(n);

Chapitre 3 · Calculs & Comparaisons

Les opérateurs

Les opérateurs permettent de faire des calculs, des comparaisons et des tests logiques sur les variables.

Opérateurs arithmétiques
Op.NomExemple
+Addition / Concaténation3 + 4 = 7
-Soustraction10 - 3 = 7
*Multiplication4 * 5 = 20
/Division10 / 2 = 5
%Modulo (reste)10 % 3 = 1
Opérateurs de comparaison
Op.Signification
==Égal à (valeur)
!=Différent de
>Supérieur à
>=Supérieur ou égal à
<Inférieur à
<=Inférieur ou égal à
Opérateurs logiques
&& — ET
Vrai si les deux conditions sont vraies.
age >= 18 && permis == true
|| — OU
Vrai si au moins une condition est vraie.
jour == "sam" || jour == "dim"
! — NON
Inverse la valeur logique.
!estConnecte → true si déconnecté
Structures de contrôle
conditions.js
// ─── if / else if / else ───
let note = 14;

if (note >= 16) {
  document.write("Très bien");
} else if (note >= 12) {
  document.write("Bien");
} else {
  document.write("À améliorer");
}

// ─── Boucle for ───
for (let i = 0; i < 5; i++) {
  document.write("i = " + i + "<br>");
}

// ─── Boucle while ───
let n = 0;
while (n < 3) {
  document.write("n = " + n + "<br>");
  n++;
}

Chapitre 4 · Fonctions & DOM

Fonctions & Manipulation du DOM

Les fonctions permettent de réutiliser du code. Le DOM (Document Object Model) permet de modifier les éléments HTML depuis JavaScript.

Les fonctions
fonctions.js
// Déclaration d'une fonction
function bonjour(prenom) {
  let message = "Bonjour, " + prenom + " !";
  return message;
}

// Appel de la fonction
let res = bonjour("Ali"); // → "Bonjour, Ali !"
document.write(res);

// Fonction avec plusieurs paramètres
function addition(a, b) {
  return a + b;
}
document.write(addition(5, 3)); // → 8
Sélectionner des éléments HTML
getElementById()
Sélectionne un élément unique par son id. Retourne null si non trouvé.
document.getElementById("titre")
getElementsByName()
Sélectionne tous les éléments ayant le même name. Retourne une liste.
document.getElementsByName("choix")
Modifier le contenu, les attributs et les styles
Propriété / MéthodeRôleExemple
element.innerHTMLLire/modifier le contenu HTML d'un élémentel.innerHTML = "<b>Texte</b>"
element.valueLire/modifier la valeur d'un champ de formulaireel.value = "Ali"
element.checkedCocher/décocher une case checkbox ou radioel.checked = true
element.disabledDésactiver / réactiver un élément de formulaireel.disabled = true
element.srcModifier la source d'une image ou vidéoel.src = "photo.jpg"
element.style.couleurModifier le style CSS d'un élément (camelCase)el.style.color = "red"
element.style.backgroundModifier la couleur de fondel.style.background = "#fff"
element.style.fontSizeModifier la taille de police (camelCase)el.style.fontSize = "20px"
element.play()Lancer la lecture d'un audio ou d'une vidéoel.play()
element.pause()Mettre en pause un audio ou une vidéoel.pause()
select.selectedIndexRécupérer l'indice de l'option sélectionnée d'un <select>sel.selectedIndex

Chapitre 5 · Objets intégrés

Chaînes de caractères & Objet Math

JS fournit des méthodes intégrées pour manipuler les textes et effectuer des calculs mathématiques.

Méthodes des chaînes de caractères
ch désigne ici un objet String (une chaîne de caractères). L'opérateur + sert à la concaténation.
Propriété / MéthodeDescriptionExemple
ch.lengthRetourne la longueur de la chaîne"Bonjour".length → 7
ch.indexOf(ch1 [,p])Position de la 1ère occurrence de ch1 (à partir de p, sinon -1)"Bonjour".indexOf("o") → 1
ch.lastIndexOf(ch1 [,p])Position de la dernière occurrence de ch1"Bonjour".lastIndexOf("o") → 4
ch.substring(d, f)Retourne une sous-chaîne de la position d à f (non incluse)"Bonjour".substring(0,3) → "Bon"
ch.replace(ch1, ch2)Remplace la 1ère occurrence de ch1 par ch2"Bonjour".replace("jour","soir") → "Bonsoir"
ch.toLowerCase()Convertit en minuscules"BONJOUR".toLowerCase() → "bonjour"
ch.toUpperCase()Convertit en majuscules"bonjour".toUpperCase() → "BONJOUR"
ch.trim()Supprime les espaces en début et fin de chaîne" Ali ".trim() → "Ali"
String.fromCharCode(n)Retourne le caractère correspondant au code ASCII nString.fromCharCode(65) → "A"
L'objet Math
Math.abs(x)
Retourne la valeur absolue (positive) de x.
Math.abs(-5) → 5
Math.sqrt(x)
Retourne la racine carrée de x.

Math.round(x)
Arrondit x à l'entier le plus proche.
Math.round(3.7) → 4
Math.trunc(x)
Retourne la partie entière de x en retirant les décimales.
Math.trunc(3.9) → 3
Math.random()
Retourne un nombre aléatoire réel dans [0, 1[.
Math.floor(Math.random()*6)+1 → dé

Chapitre 6 · Collections & Temps

Tableaux & Objet Date

Les tableaux permettent de stocker plusieurs valeurs dans une seule variable. L'objet Date permet de travailler avec les dates et heures.

L'objet Array — les tableaux
En JavaScript, un tableau se crée avec des crochets []. Les indices commencent à 0. La propriété length donne le nombre d'éléments.
tableaux.js
// Créer un tableau
let notes = [14, 18, 11, 16, 9];

// Accéder à un élément (indice 0 = premier)
document.write(notes[0]); // → 14
document.write(notes[2]); // → 11

// Nombre d'éléments
document.write(notes.length); // → 5

// Parcourir avec une boucle
for (let i = 0; i < notes.length; i++) {
  document.write("Note["+i+"] = " + notes[i] + "<br>");
}
L'objet Date
new Date()
Crée un objet Date représentant la date et heure actuelles.
let d = new Date();
new Date(ch)
Crée un objet Date à partir d'une chaîne représentant une date.
let d = new Date("2025-06-15");
d.getDate()
Retourne le jour du mois (entre 1 et 31).
d.getDate() → 15
d.getMonth()
Retourne le numéro du mois (entre 0 et 11 — attention !).
d.getMonth() + 1 → mois réel
d.getFullYear()
Retourne l'année sur 4 chiffres.
d.getFullYear() → 2025
d.toString()
Retourne la date sous forme de chaîne de caractères lisible.
d.toString() → "Mon Jun 15..."

Chapitre 7 · Réactivité

Les événements HTML

Un événement est une action déclenchée par l'utilisateur. JavaScript écoute ces actions pour y répondre en temps réel.

Les événements s'écrivent comme attributs HTML (onclick="maFonction()") ou via addEventListener en JS pur.
Tableau complet des événements
ÉvénementSe déclenche quand…Utilisé sur
onclickL'utilisateur clique sur l'élémentBoutons, liens, images…
onmouseoverLe pointeur survole l'élémentMenus, cartes, images
onmouseoutLe pointeur quitte l'élémentMenus, survols
onfocusL'élément obtient le focus (clavier/clic)Champs de formulaire
onblurL'élément perd le focusChamps de formulaire
onchangeLa valeur change ET l'élément perd le focusSelect, input, textarea
oninputLa valeur change en temps réel (sans attendre blur)Input text, textarea
onsubmitLe formulaire est soumisBalise <form>
onkeydownUne touche du clavier est presséeInput, document
onkeyupUne touche du clavier est relâchéeInput, document
onloadLa page (ou une image) est complètement chargée<body>, <img>
onplayLa lecture d'un média commence<audio>, <video>
onpauseUn média est mis en pause<audio>, <video>
