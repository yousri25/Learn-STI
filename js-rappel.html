③Méthodes d'affichage et de saisie

Pour afficher ou modifier un contenu dynamique, on peut utiliser la propriété innerHTML d'un élément HTML.
MéthodeDescriptionExemple
alert(message) Affiche un message dans une boîte de dialogue. alert("Bonjour !");
document.write() Affiche directement dans le document HTML. document.write("<p>Texte</p>");
prompt(message) Affiche une boîte de dialogue avec une zone de saisie et retourne la valeur saisie. let n = prompt("Votre nom ?");
element.innerHTML Permet d'afficher ou modifier le contenu HTML d'un élément. document.getElementById("d").innerHTML = "Texte";

④Méthodes pour sélectionner des éléments HTML

MéthodeDescriptionExemple
document.getElementById(id) Permet de sélectionner un élément unique en fonction de son identifiant. document.getElementById("titre")
document.getElementsByName(name) Permet de sélectionner tous les éléments ayant le même attribut name. document.getElementsByName("choix")

⑤Modification du contenu, des attributs et des styles HTML

Se limiter à la manipulation des attributs : value, checked, disabled, readonly, src, muted.
SyntaxeDescriptionExemple
element.attribut = valeur Change la valeur de l'attribut d'un élément HTML. img.src = "photo.png";
element.style.propriété = valeur Change le style CSS d'un élément HTML. Se limiter à : color, background, border, font, width, height. el.style.color = "red";
element.méthode() Effectue une action sur un élément HTML. Se limiter aux méthodes play() et pause(). video.play();
select.selectedIndex Pour une liste déroulante <select>, retourne l'indice de l'option sélectionnée. let i = sel.selectedIndex;

⑥Fonctions / Méthodes JavaScript

* Si la conversion n'est pas possible, les fonctions de conversion retournent NaN.
FonctionDescriptionExemple
isNaN(ch) Retourne true si l'argument ch n'est pas convertible en un nombre, sinon retourne false. isNaN("abc") → true
Number(ch) Convertit une chaîne ch en un nombre (entier ou réel).* Number("3.14") → 3.14
parseFloat(ch) Convertit une chaîne ch en un réel.* parseFloat("3.14") → 3.14
parseInt(ch [,b]) Convertit une chaîne ch en un entier exprimé dans la base b.* parseInt("10", 2) → 2
String(a) Convertit la valeur a en une chaîne de caractères. String(42) → "42"

⑨Les chaînes de caractères

ch est un objet String. L'indexation des caractères commence à 0.
Méthode / PropriétéDescriptionExemple
+ Opérateur de concaténation. "Bon" + "jour" → "Bonjour"
ch.length Propriété qui retourne la longueur de ch. "Bonjour".length → 7
ch.indexOf(ch1 [,p]) Retourne la position de la 1ère occurrence de ch1 dans ch, en effectuant la recherche à partir de la position p (sinon -1 si non trouvé). "Bonjour".indexOf("o") → 1
ch.lastIndexOf(ch1 [,p]) Retourne la position de la dernière occurrence de ch1 dans ch à partir de la position p. "Bonjour".lastIndexOf("o") → 4
ch.substring(d, f) Retourne une sous-chaîne de ch de la position d à la position f (non incluse). "Bonjour".substring(0,3) → "Bon"
ch.replace(ch1, ch2) Retourne une chaîne dans laquelle la 1ère occurrence de ch1 dans ch est remplacée par ch2. "Bonjour".replace("o","0") → "B0njour"
ch.toLowerCase() Convertit tous les caractères de ch en minuscules. "HELLO".toLowerCase() → "hello"
ch.toUpperCase() Convertit tous les caractères de ch en majuscules. "hello".toUpperCase() → "HELLO"
ch.trim() Supprime tous les espaces existants au début et à la fin de ch. " Bonjour ".trim() → "Bonjour"
String.fromCharCode(num, …, numN) Retourne une chaîne formée par la concaténation des résultats de conversion des codes ASCII passés en paramètres. String.fromCharCode(65) → "A"

⑩L'objet Date

d est un objet Date. Les mois sont numérotés de 0 (janvier) à 11 (décembre).
Constructeur / MéthodeDescriptionExemple
new Date() Retourne la date et heure courante. let d = new Date();
new Date(ch) Retourne un objet Date à partir d'une chaîne de caractères représentant une date. new Date("2024-09-01")
d.getDate() Retourne le jour du mois (entre 1 et 31). d.getDate() → 15
d.getMonth() Retourne le numéro du mois (entre 0 et 11). d.getMonth() → 8 // septembre
d.getFullYear() Retourne la valeur de l'année sur 4 chiffres. d.getFullYear() → 2024
d.toString() Retourne une chaîne de caractères représentant la date d. d.toString() → "Sun Sep 01 2024…"
