Apprendre PHP

Un guide pas à pas pour créer des pages web dynamiques côté serveur

Chapitre 1 · Introduction

C'est quoi PHP ?

HTML, CSS et JavaScript s'exécutent dans le navigateur. PHP, lui, s'exécute sur le serveur — avant que la page soit envoyée au visiteur.

// Définition

PHP signifie Hypertext Preprocessor. C'est un langage de programmation côté serveur qui génère du HTML dynamique. Il peut recevoir des données d'un formulaire, communiquer avec une base de données MySQL, et renvoyer une page HTML personnalisée au navigateur.

💡 Le pont entre front-end et base de données

HTML/CSS/JS → s'exécutent dans le navigateur (côté client).
PHP → s'exécute sur le serveur Apache (via XAMPP).
MySQL → stocke les données sur le serveur.

PHP est le pont : il reçoit les données du formulaire HTML, interroge MySQL, et génère la page HTML de réponse.

Comment fonctionne PHP ?
Fichier .php
Un fichier PHP contient du code PHP mélangé avec du HTML. Il a l'extension .php et doit être placé dans le dossier htdocs de XAMPP.

Le code PHP doit être encadré par des balises d'ouverture et fermeture. Tout ce qui est en dehors est du HTML normal.
<?php
  // code PHP ici
?>

<?php echo "<b>Texte gras</b>"; ?>

Chapitre 2 · Les données

Variables & Types de données

En PHP, toutes les variables commencent par le symbole $. PHP est typé dynamiquement — le type est déterminé automatiquement selon la valeur.

Déclarer et utiliser une variable
variables.php
<?php
// Déclaration — toutes les variables commencent par $
$prenom = "Ali";        // string
$age    = 17;          // int
$note   = 18.5;        // float
$reussi = true;        // bool
$vide   = null;        // null
$notes  = [14, 18, 11]; // array

// Affichage avec echo
echo "Bonjour " . $prenom . ", tu as " . $age . " ans.";

// Le . est l'opérateur de concaténation en PHP
?>
Les types de données PHP

intEntierNombre entier positif ou négatif$age = 17;
floatRéelNombre décimal$note = 18.5;
stringChaîneTexte entre guillemets simples ou doubles$nom = "Ali";

Opérateurs de transtypage
Le transtypage (cast) force la conversion d'une variable d'un type à un autre.
(int)
Convertit en entier.
(int)"42.9" → 42
(float)
Convertit en nombre réel.
(float)"3.14" → 3.14
(string)
Convertit en chaîne de caractères.
(string)42 → "42"
(bool)
Convertit en booléen (0, "", null → false, le reste → true).
(bool)1 → true
(array)
Convertit en tableau.
(array)"Ali" → ["Ali"]

Chapitre 4 · Fonctions

Fonctions & Chaînes de caractères

PHP dispose de centaines de fonctions intégrées. Voici les plus utiles pour les chaînes de caractères et les maths.

Déclarer et appeler une fonction
fonctions.php
<?php
// Déclaration d'une fonction
function calculerMoyenne($notes) {
    $somme = 0;
    foreach ($notes as $n) {
        $somme += $n;
    }
    return $somme / count($notes);
}

// Appel de la fonction
$notes = [14, 18, 11, 16];
$moy   = calculerMoyenne($notes);
echo "Moyenne : " . $moy;   // → 14.75
?>

Chapitre 5 · Collections & Temps

Tableaux & Fonctions de date

Les tableaux PHP peuvent être indexés numériquement ou associatifs. Les fonctions de date permettent de manipuler les dates et heures.

Les tableaux PHP
tableaux.php
<?php
// Tableau indexé (numérique)
$notes = [14, 18, 11, 16];
echo $notes[0];       // → 14 (indice commence à 0)
echo count($notes);   // → 4 (nombre d'éléments)

// Parcourir avec foreach
foreach ($notes as $n) {
    echo $n . " ";
}

// Tableau associatif (clé => valeur)
$eleve = [
    "nom"    => "Ali",
    "age"    => 17,
    "classe" => "Bac STI"
];
echo $eleve["nom"];    // → Ali
echo $eleve["classe"]; // → Bac STI

// Créer un tableau avec array()
$couleurs = array("rouge", "vert", "bleu");
?>
Fonctions sur les tableaux
count($t)
Retourne le nombre d'éléments dans le tableau.
count([1,2,3]) → 3
array($v1, $v2…)
Crée un tableau à partir d'une liste de valeurs.
array("a","b","c")
Fonctions de Date et Heure
FonctionDescriptionExemple
date($format)Retourne la date actuelle formatée selon $formatdate("d/m/Y") → "15/06/2025"
time()Retourne l'heure actuelle en timestamp (secondes depuis 01/01/1970)time() → 1750000000

Chapitre 6 · Interaction avec l'utilisateur

Formulaires & Variables superglobales

PHP reçoit les données envoyées par les formulaires HTML via les superglobales $_POST et $_GET.

// Le cycle formulaire → PHP

1. L'utilisateur remplit un formulaire HTML et clique Envoyer.
2. Le navigateur envoie les données au serveur via HTTP POST ou GET.
3. PHP reçoit ces données dans $_POST['nom_champ'] ou $_GET['nom_champ'].
4. PHP les traite et génère la réponse HTML.

$_POST et $_GET
$_POST
Reçoit les données envoyées via method="post". Les données ne sont pas visibles dans l'URL. Utilisé pour les données sensibles (mots de passe, formulaires).
$nom = $_POST['nom'];
$email = $_POST['email'];
$_GET
Reçoit les données envoyées via method="get" ou passées dans l'URL. Les données sont visibles dans l'URL. Utilisé pour les recherches, filtres.
// URL: page.php?id=5
$id = $_GET['id']; // → 5
Exemple complet : formulaire → traitement PHP
formulaire.php
<!-- ── formulaire.html ── -->
<form action="traitement.php" method="post">
  <label>Nom : <input type="text" name="nom" required></label>
  <label>Note : <input type="number" name="note" min="0" max="20"></label>
  <input type="submit" value="Envoyer">
</form>

<!-- ── traitement.php ── -->
<?php
// 1. Récupérer les données du formulaire
$nom  = $_POST['nom'];
$note = (float) $_POST['note'];

// 2. Traiter et afficher
if ($note >= 10) {
    echo "<p>Félicitations " . $nom . " ! Note : " . $note . "/20</p>";
} else {
    echo "<p>" . $nom . ", note insuffisante : " . $note . "/20</p>";
}
?>
Fonctions diverses utiles
isset($var)
Vérifie si une variable est définie et non nulle. Indispensable avant d'utiliser $_POST.
if (isset($_POST['nom'])) { ... }
die($msg)
Affiche $msg et arrête immédiatement l'exécution du script. Utilisé pour les erreurs critiques.
die("Connexion impossible !");
require($fichier)
Inclut et exécute le fichier spécifié. Si le fichier n'existe pas, génère une erreur fatale.
require("connexion.php");
echo
Affiche une ou plusieurs expressions. C'est la commande principale pour générer du HTML.
echo "Bonjour " . $nom;

Chapitre 7 · Connexion à la base de données

PHP & MySQL (MySQLi)

PHP communique avec MySQL via l'extension MySQLi. C'est la brique centrale d'un site dynamique : lire, insérer, modifier des données.

// Le cycle PHP ↔ MySQL

1. PHP ouvre une connexion à MySQL avec mysqli_connect().
2. PHP envoie une requête SQL avec mysqli_query().
3. PHP récupère les résultats ligne par ligne avec mysqli_fetch_array().
4. PHP ferme la connexion avec mysqli_close().

Fonctions MySQLi
FonctionDescription
mysqli_connect($h,$u,$p,$db)Établit une connexion à la base de données MySQL. Retourne un objet connexion ou false.
mysqli_query($conn, $sql)Envoie une requête SQL à la base. Retourne un résultat ou false.
mysqli_fetch_array($res)Retourne une ligne du résultat sous forme de tableau associatif et numérique.
mysqli_fetch_row($res)Retourne une ligne du résultat sous forme de tableau numérique uniquement.
mysqli_num_rows($res)Retourne le nombre de lignes retournées par un SELECT.
mysqli_affected_rows($conn)Retourne le nombre de lignes affectées par INSERT, UPDATE ou DELETE.
mysqli_error($conn)Retourne le message d'erreur de la dernière requête SQL exécutée.
mysqli_close($conn)Ferme la connexion à la base de données.
Exemple complet : SELECT avec PHP
liste_eleves.php
<?php
// 1. Connexion à MySQL
$conn = mysqli_connect("localhost", "root", "", "ecole");

if (!$conn) {
    die("Connexion échouée : " . mysqli_connect_error());
}

// 2. Envoyer la requête SQL
$sql = "SELECT nom, prenom, note FROM eleve ORDER BY note DESC";
$res = mysqli_query($conn, $sql);

// 3. Lire et afficher les résultats
echo "<table border='1'><tr><th>Nom</th><th>Prénom</th><th>Note</th></tr>";

while ($ligne = mysqli_fetch_array($res)) {
    echo "<tr>";
    echo "<td>" . $ligne['nom']    . "</td>";
    echo "<td>" . $ligne['prenom'] . "</td>";
    echo "<td>" . $ligne['note']   . "</td>";
    echo "</tr>";
}
echo "</table>";

// 4. Fermer la connexion
mysqli_close($conn);
?>
Exemple : INSERT depuis un formulaire
ajouter_eleve.php
<?php
if (isset($_POST['nom'])) {

  // Récupérer les données du formulaire
  $nom       = $_POST['nom'];
  $prenom    = $_POST['prenom'];
  $note      = (float) $_POST['note'];
  $id_classe = (int)   $_POST['id_classe'];

  // Connexion
  $conn = mysqli_connect("localhost", "root", "", "ecole");

  // Construire et exécuter l'INSERT
  $sql = "INSERT INTO eleve (nom, prenom, note, id_classe)
          VALUES ('" .$nom."','".$prenom."',".$note.",".$id_classe.")";
  $res = mysqli_query($conn, $sql);

  if ($res) {
      echo "Élève ajouté avec succès !";
  } else {
      echo "Erreur : " . mysqli_error($conn);
  }
  mysqli_close($conn);
}
?>
